# Derivatives

* calculate symbolic differentiation from symbolic expressions
* supporting functions
    * `Number` (0, 1, 2, ...)
    * `Variable` ('x', 'y', 'z', ...)
    * `Addition` (e1 + e2)
    * `Subtraction` (e1 - e2)
    * `Multiplication` (e1 * e2)
    * `Division` (e1 / e2)
    * `ConstantExponentiation` (exp b e)
    * `Logarithm` (log c e)
* simple rules for reduction
